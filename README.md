
## 1. Instalação das dependências

```bash
pip install -r requirements.txt
```

2. Execute migração:

```bash
python manage.py makemigrations
python manage.py migrate
```

3. Inicie o servidor:

```bash
python manage.py runserver
```

4. Acesse em http://localhost:8000/api/livros/
